SYE Update Log

Version 1.1 is live. It brings the new navigation, the new feature layout, and a clearer focus on what SYE is for — your eyes.

Version 1.1

Released: March 2026

Current version

What's new

  • Tab Bar Navigation with Home, Functions, Insights, and Settings.
  • Block Scheduling with up to 3 daily recurring schedules and live notifications.
  • Quick App Blocks, Expected Phone Use, and Screen Time Alert moved to the Home tab.
  • Preset customisation for Quick App Blocks so any slot can be reassigned.
  • First-launch setup flow with clear permission guidance and preview screens.
  • Permissions card with live status, step-by-step prompts, and open-settings shortcuts.
  • iPad support with two-column layouts and centred card layouts.
  • Rate, support, and update log access added to Settings.

Bug fixes

  • Fixed Screen Time alert firing below the usage threshold.
  • Fixed Screen Time gate flashing for returning approved users.
  • Fixed "Yes I need it" not opening the app in some cases.
  • Fixed soft interruption shield showing the wrong screen.
  • Fixed preset customisation triggering on every launch.
  • Fixed duplicate notification cards on the Home screen.
  • Fixed expected phone use loading indefinitely when DeviceActivity had no data.
  • Fixed schedule overlap detection incorrectly flagging the first schedule.

Known issues in 1.1

  • First tap on a preset after app restart may show a blank sheet; tap again to resolve it.
  • Block timers are accurate to within 1 to 5 minutes due to Apple's DeviceActivity framework.
  • Screen Time report data can take up to 24 hours to appear on first install.
  • Selecting whole app categories may not block all apps reliably; individual apps are best.
  • Previously selected apps cannot be displayed in Block Schedule edit view, but selections remain active.
Version 1.0

Released: March 25, 2026

Initial release of SYE — Save Your Eyes.

What was included

  • Quick App Blocks with five customisable preset slots and independent timers.
  • Soft Interruptions with pause screens, quotes, and streak tracking.
  • Expected Phone Use in Automatic and Manual modes.
  • Today's Screen Time using Apple's DeviceActivity report.
  • Screen Time Alert for daily usage threshold notifications.
  • Privacy-first design with no servers, no accounts, and no data collection.

Known issues fixed in 1.1

  • Screen Time alert fired regardless of actual usage.
  • Screen Time gate appeared briefly for returning users.
  • Soft interruption shield occasionally showed the wrong screen.
